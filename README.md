# README

Replication code. Integrates LandVote, LCV, and U.S. election data.

For details on data processing, please refer to the our code for [LandVote](https://github.com/cassiebuhler/datasets/tree/main/landvote), [census boundaires](https://github.com/cassiebuhler/datasets/tree/main/census), and [political party](https://github.com/cassiebuhler/datasets/tree/main/political_parties).

#### Data sources

- TPL LandVote Database by Trust for Public Land. Data: https://tpl.quickbase.com/db/bbqna2qct?a=dbpage&pageID=8. Citation: The Trust for Public Land, LandVote®, 2025, www.landvote.org., License: Public Domain

- MIT Election Data and Science Lab. (2017). U.S. President 1976-2020 (Version V8) [dataset]. Harvard Dataverse. https://doi.org/10.7910/DVN/42MVDX 

- MIT Election Data and Science Lab. (2018). County Presidential Election Returns 2000-2024 (Version V16) [dataset]. Harvard Dataverse. https://doi.org/10.7910/DVN/VOQCHQ

- Federal Election Commission. (2025, January 16). Official 2024 presidential general election results. https://www.fec.gov/resources/cms-content/documents/2024presgeresults.xlsx

- Leip, David. Dave Leip’s Atlas of U.S. Presidential Elections. http://uselectionatlas.org. Note: Used for 1988-1996 County Presidential Election Returns.

- U.S. Census Bureau. (2020). State FIPS Codes (2020). https://www2.census.gov/geo/docs/reference/codes2020/national_state2020.txt. Used to identify and map U.S. states via Federal Information Processing Standards (FIPS) codes.

- U.S. Census Bureau. (2025). TIGER/Line Shapefiles: Places (Incorporated Places and Census Designated Places). https://www2.census.gov/geo/tiger/TIGER2025/PLACE/ Used to obtain spatial boundaries for all Census-designated places across the 50 U.S. states.

- U.S. Census Bureau. (2020). County Subdivision (County Subdivision Codes). https://www2.census.gov/geo/docs/reference/codes2020/national_cousub2020.txt. Used to identify and map county subdivisions (e.g., townships, towns, precincts) within states.

- U.S. Census Bureau. (2024). Counties (County Boundaries and Reference Data).
TIGER/Line Shapefiles, Counties. https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html Used to obtain county-level geographic boundaries and identifiers.
